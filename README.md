AI Dev Toolkit
A lightweight developer utility built with React and TypeScript, powered by the Anthropic Claude API.
Features
	•	Code Explainer — Paste any code snippet and get a clear, step-by-step explanation
	•	Text Summarizer — Summarize long documentation or articles into key points
	•	Bug Finder — Analyze code for bugs and get fix suggestions
Tech Stack
	•	React + TypeScript
	•	Vite
	•	Anthropic Claude API
Getting Started
	1.	Clone the repo
  git clone https://github.com/yourusername/ai-dev-toolkit.git
cd ai-dev-toolkit

2.	Install dependencies
   npm install
3.	Add your Anthropic API key
Create a .env file in the root:
VITE_ANTHROPIC_API_KEY=your_key_here
4.	Run the app
   npm run dev
Usage
Select a tool from the top navigation, paste your input, and click Run. The AI will process your request and display the result below.
